# Lista de Tarefas

## Descrição
Uma aplicação web simples para gerenciar uma lista de tarefas. Permite adicionar, visualizar e gerenciar tarefas de forma intuitiva.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## Como Usar
1. Abra o arquivo `index.html` em um navegador web.
2. Digite uma tarefa no campo de entrada.
3. Clique em "Adicionar" para incluir a tarefa na lista.
4. As tarefas cadastradas aparecerão na lista abaixo.

## Estrutura do Projeto
- `index.html`: Arquivo principal da aplicação.
- `CSS/style.css`: Estilos da página.
- `JS/script.js`: Lógica JavaScript para funcionalidade da lista de tarefas.
